## About

I'm 晓灰, a "cyber farmer" who writes code and words. My work sits at the intersection of AI agents, software engineering, knowledge workflows, and product systems.

- Agent expert at Ant Group, working on production agent systems for insurance scenarios.
- Lead agent work for a million-MAU insurance product, including product interpretation, quality scoring, and evaluation pipelines.
- Creator of [MCPAdvisor](https://github.com/istarwyh/mcpadvisor), a tool for discovering and installing MCP servers.
- Built `writingHelper`, which has served **15,000+ users**.
- PKU '21. Based in Beijing.

## What I Build

- **Agent-native products**: systems where agents close the loop between user intent, production workflows, evaluation, and product improvement.
- **MCP and tool ecosystems**: practical ways to select, route, and compose external capabilities for agents.
- **Context engineering**: memory, retrieval, state, event streams, and runtime patterns that make agents more reliable.
- **Digital gardens and writing systems**: tools and habits for turning scattered notes into public, useful knowledge.
